# Instructions

## Install Python

## Initiate and run venv
```bash
python -m venv venv
.\venv_name\Scripts\activate
```

## Install python modules from requirements.txt
```bash

python -m pip install -r requirements.txt
```

## Obtain data
```bash
python 01-download_data.py
```

## Process data
```bash
python 02-process_data.py
```

## Build Dashboard
```bash
python 03-build_dashboard.py
```
